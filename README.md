# electron-rpc
Eloctron RPC Discord ile çalışan bir rpc paketi denilebilir panelli düzenleyebildiğiniz bir rpc
Sürüm beta 0.0.1



## kurulum
projeyi clonla 

```
$ git clone https://github.com/cenapyuce/electron-rpc.git
```
Mödülleri yüklemek 1. adım
```
$ npm install
```
2. Adım Başlat
```
$ npm start
```

3. Keyfini cıkar
